# LeetCode Solutions 🧠💻

This repository contains my curated solutions to various **LeetCode** problems, written in **Python**, **Java**, **PHP**, **JavaScript**, **Go**, **Rust**.

Each solution is:
- Categorized by **difficulty level**: `Easy`, `Medium`, `Hard`
